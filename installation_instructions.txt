All required files have been provided in a zip folder titled Wafiqah_Raisa_WMGAssignment. To install the contents of the zip folder, please follow the instructions below. All code has been written using the Python programming language. 

1. Unzip the folder
2. Create a virtual environment that can run Python version 3.9.6
3. Run the following command in the terminal/command line to install dependencies/libraries: pip install -r requirements.txt
4. Run the following command in the terminal/command line to execute the code: python3 -i main.py

Please note that upon running the code, that 3 Matplotlib figures will be outputted and the answer to Question 5 will be printed to stdout.
